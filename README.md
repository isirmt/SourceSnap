## Development

To install the module, run the following in the root directory.

```bash
pnpm i
```

To run the development environment,

```bash
pnpm dev
```

To build the local environment,

```bash
pnpm build
```

To run lint,

```bash
pnpm check
# or
pnpm lint
pnpm lint:fix
pnpm fmt
```
